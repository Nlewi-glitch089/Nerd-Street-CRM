**Disabled-signin protection & admin notification**

- What it does: Blocks sign-in attempts from deactivated users and records each blocked attempt as an `ActionLog` entry (`action = signin_attempt_disabled`). Adds an admin notification bell that surfaces recent disabled-signin attempts and a small panel showing details (email, time, reason).
- Why: Prevents deactivated users from authenticating while giving admins visibility into attempted logins for security and auditing.
- Key behaviors:
